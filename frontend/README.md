## Требования

Перед запуском проекта убедитесь, что у вас установлен Bun.

### Установка Bun

Официальная документация:

- https://bun.sh/docs/installation

Проверить установленную версию:

```bash
bun --version
```

---

## Установка зависимостей

После клонирования репозитория установите зависимости:

```bash
bun install
```

---

## Запуск проекта

Для запуска frontend-приложения в режиме разработки выполните команду:

```bash
bun run dev
```

После успешного запуска приложение будет доступно по адресу:

```txt
http://localhost:5173
```
